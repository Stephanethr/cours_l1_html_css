Introduction au Développement Web

HTML, CSS et JavaScript

Objectifs du cours

  • Comprendre l'organisation d'un site web
  • Créer des pages web structurées avec HTML
  • Styliser des pages avec CSS
  • Dynamiser une page web avec JavaScript
  • Rendre un site web responsive

Comment fonctionne un site web ?

Un site web est composé de fichiers HTML, CSS et JavaScript

Structure d'un projet Web


/mon-projet
 ├── index.html
 ├── styles.css
 ├── script.js
 ├── images/
 ├── assets/

HTML : Structure d'une page





    
    Mon Site

CSS : Styliser une page


body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
}
h1 {
    color: blue;
}

JavaScript : Dynamiser une page


document.querySelector('h1').addEventListener('click', function() {
    alert('Titre cliqué !');
});

Media Queries : Rendre une page responsive


@media screen and (max-width: 768px) {
    body {
        background-color: lightgrey;
    }
}
